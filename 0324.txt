## 4. Python 3.12 弯引号兼容性修复（追加）

src/runner/run_case.py
- dict key "A01" 本身使用了 U+201C/U+201D 弯引号，Python 3.12 将其识别为字符串定界符导致 SyntaxError
- 用脚本全量替换文件内所有 U+201C / U+201D 为 ASCII 双引号


## 5. Actor 重复输出 bug 修复（三处）

问题根因：actor 每轮看到的 prompt 完全相同，低 temperature 下输出 100% 重复。

src/actor/engine.py
- Bug 修复：user_prompt 中补充"可选的下一状态"列表（含各 state 的 description）
  原因：prompt 里只有 allowed_actions，没有合法 states 列表，
        导致 Qwen 输出的 state 字段无效，被 fallback 到 current_state，state 机器失效

src/runner/run_case.py（Phase 3 循环 + Phase 4 循环，各一处）
- Bug 修复：删除 runner 里对 next_state 的重复过滤
  原因：actor/engine.py 内部已做 state 验证，runner 里再过滤一次
        导致即使 actor 输出了合法 state，也可能被错误地打回 current_state
